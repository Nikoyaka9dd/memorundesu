自分が欲しいがために作っただけの自己満ウィジェットです。

走り書きでメモりたい時に使います。

## Install

以下からdmgファイルをダウンロードしてインストールしてください。

https://github.com/Nikoyaka9dd/memorundesu/releases/tag/v1.0.0

## How to use

マークダウンで書けます。

ダブルクリックで編集モード、もう一度ダブルクリックすると表示モードになります。
